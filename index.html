Official Open Source Effort

FinOps that moves from insight to action.

FinOps-as-Code is an open source effort aligned with the FinOps Foundation ecosystem. It focuses on practical automation that helps teams act on cloud cost, efficiency, and governance opportunities through scripts and executable workflows.

Manifest

Visibility matters. Execution changes outcomes.

The work here is built for teams that already know where waste exists and now need code they can inspect, test, and run to do something about it.

  • Scripts first
  • Safe defaults
  • Real FinOps use cases
  • Open source quality bar

Why This Exists

A practical home for reusable FinOps automation.

The current repositories focus on concrete operational problems: idle resources, avoidable cloud waste, and the gap between cost visibility and actual follow-through.

The documentation exists to make that work easier to adopt and easier to extend. It explains how to contribute, what engineering standards matter, and what kinds of automation fit the project well.

Read the initiative, then contribute with code that others can actually run.

Open Source Direction

The current repositories are examples, not the boundary.

New repositories can be created whenever a real FinOps problem deserves its own automation. What should remain consistent is the contribution bar: practical use case, safe execution, readable code, and value that other teams can reuse.
