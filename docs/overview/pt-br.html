Visão Geral

Documentação Compartilhada para FinOps-as-Code

Este repositório fornece a camada pública de documentação da organização FinOps-as-Code, dando aos projetos um lugar comum para orientação, padrões e crescimento futuro da documentação.

English Version

Propósito

  • Fornecer uma home pública de documentação via GitHub Pages.
  • Centralizar orientação reutilizável entre os repositórios da FinOps-as-Code.
  • Apoiar contribuidores com expectativas compartilhadas e boas práticas de engenharia.
  • Manter a documentação leve enquanto a organização cresce.

Direção Atual

O foco atual da organização continua sendo automação prática de FinOps, especialmente scripts executáveis e independentes. A documentação existe para tornar essas contribuições mais fáceis de entender, adotar e revisar.

Por que um Repositório Separado de Docs

  • Cria uma casa estável para orientação em nível de organização.
  • Evita duplicar a mesma documentação central em cada repositório.
  • Suporta GitHub Pages sem acoplar o site a um único projeto de automação.
  • Deixa espaço para padrões futuros, notas de governança e exemplos compartilhados.

Escopo do Repositório Hoje

Hoje este repositório de documentação é intencionalmente simples. Ele inclui uma home pública, pontos de entrada para docs, guias compartilhados de contribuição e padrões básicos. Com o tempo, pode evoluir para um site mais completo se isso fizer sentido para a organização.
